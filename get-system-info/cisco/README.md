# CISCO 获取系统信息指令
获取系统信息的指令可以在对应的文件夹中看到。
  
常见的 **`CISCO`** 路由器和交换机获取系统信息指令:  
1. `en`
2. `sh ver`
3. `sh inv`
4. `sh env all`
5. `sh diag`
6. `sh post`
7. `sh lic`
8. `sh lic udi`
  
常见的  **`CISCO NEXUS`** 交换机获取系统信息指令:  
1. `en`
2. `sh ver`
3. `sh inv`
4. `sh module`
5. `sh env power`
6. `sh env fan`
7. `sh env temp`
8. `sh diag result all`
9. `sh lic usage`
10. `sh lic`
  
不同型号的指令可能略有不同。
